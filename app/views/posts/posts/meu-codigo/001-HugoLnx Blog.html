Introdução

Este blog foi um dos meus primeiros projetos, uma documentação de como foi o desenvolvimento do mesmo pode ser visualizada no Google Docs(Esta ultima release não está inclusa).

Inicialmente o projeto era privado, utilizava o Assembla, porém agora o código está aberto no Github.

Principais recursos que estão sendo utilizados

  • Ruby 1.9.2
  • Rails 3.0.6
  • RSpec 2
  • Cucumber
  • History API(HTML5)
  • Novas tags do HTML5
